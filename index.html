Traveris

02

HoopStats

+20 usuarios activos

Plataforma analítica de estadísticas de la NBA en tiempo real.

Front ↗ Back ↗ Ver aplicación ↗

EL PROBLEMA

Los fanáticos y analistas de basketball tenían que buscar estadísticas en fuentes dispersas. Faltaba una solución unificada con datos dinámicos y accionables.

LA SOLUCIÓN

Un pipeline de datos que consume feeds en vivo de la NBA, procesado por un backend optimizado de Node.js, y servido a dashboards visuales en tiempo real.

CARACTERÍSTICAS CLAVE

▸ Pipeline de consumo de datos en vivo.
▸ API RESTful optimizada para baja latencia.
▸ Visualización de datos interactiva en Angular.
▸ Backend escalable en Hostinger.
Angular Node.js PostgreSQL REST APIs
